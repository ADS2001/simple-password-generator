This is a simple Python script that creates random passwords. It checks that each password is at least 8 characters long and uses a mix of letters, numbers, and special symbols to make passwords more secure.

* The input validation step rejects passwords shorter than 8 characters to help prevent weak passwords.

* The character pool includes uppercase and lowercase letters, digits, and standard ASCII symbols.

* The code uses the if __name__ == "__main__": entry point, so it can be run directly or imported as a module.
